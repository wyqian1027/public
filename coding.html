Packages

  • stats-lib
    Fast and light-weight statistical JavaScript package of frequently used operations on arrays.
    In particular, stats-lib also supports zScore, quartile, nLargest, nSmallest operations on arrays, and linear fit, covariance, correlation calculations on multiple arrays. Dec 2018. 

    //installation with Node Package Manager (NPM)
npm install stats-lib

//how to use
var stats = require('stats-lib');
var X = [4, 4, 3, 2, 1, 0];
console.log(stats.quartile(X));     //[ 1, 2.5, 4 ]
console.log(stats.kthMoment(X, 2));  //1, standardized moment of degree 2
