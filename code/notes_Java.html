  • Enums

    An enum represents a group of constants (public static final variables). Uppercase letters should be used. To create an enum:

    enum level {
    LOW, 
    MEDIUM,
    HIGH
}

    Access an enum:

    Level myVar = level.MEDIUM;

    In a switch statement:

    switch(myVar) {
    case LOW:
        System.out.println("Low level");
        break;
    case MEDIUM:
        System.out.println("Medium level"); 
        break;
    case HIGH:
        System.out.println("High level");
        break;
}
