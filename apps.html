Database Apps

  • ECOsystem, database for animals and plants at home

    Mobile-friendly RESTful web application using Node.js (Express framework) that allows users to keep track of local animals and plants. Applicatoin data efficiency is optimized since migrating from MySQL to MongoDB. Additional features includes user authentication, user authorization, and responsive Google API.

  • MyarXiv, engine for academic papers

    Fast, lightweight web application to search for academic papers, mainly in physics and mathematics, with the help of arXiv API by Cornell University. LocalStorage is used to save user data entries and subscriptions.

Static Apps

  • Weather Assistant

    Simple weather app for checking current weather at your current location, using OpenWeatherMap API. Predictions are made for the coming 3 days. Wind chill Index are calculated using the latest formula developed by Joint Action Group for Temperature Indices.
