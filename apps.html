Database Apps

  • ECOsystem -- keep a record of animals and plants at home

    Mobile-friendly RESTful web application using Node.js (Express framework) that allows users to keep track of local animals and plants. Applicatoin data efficiency is optimized since migrating from MySQL to MongoDB. Additional features includes user authentication, user authorization, and responsive Google API.

    Launch Application...Wait a few seconds for servers to start

  • MyarXiv -- search engine for academic papers

    Fast, lightweight web application to search for academic papers, mainly in physics and mathematics, with the help of arXiv API by Cornell University. LocalStorage is used to save user data entries and subscriptions.

    Launch Application...Wait a few seconds for servers to start

  • ItemizeMe -- customized tags and notes

    Beautifully-crafted Python Django web application to assist users in customizing tags and notes. The idea comes from Google Keep. Much diversed color schemes are implemented, which offers a more lively user experience. Responsive Material Design is used. Many more features to come in the future.

    Launch Application...Wait a few seconds for servers to start

Static Apps

  • Weather Assistant

    Simple weather app for checking current weather at your current location, using OpenWeatherMap API. Predictions are made for the coming 3 days. In particular, Wind chill Index is calculated using the latest formula developed by Joint Action Group for Temperature Indices.
